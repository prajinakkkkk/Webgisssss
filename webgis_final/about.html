About the Application

This application, "Healthcare Accessibility Analysis with Map Functionalities and Geospatial Analysis", is a dynamic mapping and visualization tool designed to support healthcare accessibility analysis and geospatial data exploration. Built using Leaflet.js and integrated with GeoServer, GeoJSON and external APIs, it provides the following key features:

  • Base Map Layers: Includes multiple map layers like OpenStreetMap, Satellite View and Terrain View for diverse visualization perspectives.
  • Geospatial Data Integration: - Displays Nepal's districts, airports, and healthcare facilities using GeoJSON files. - Includes a WMS layer from GeoServer for administrative boundaries.
  • Healthcare Accessibility: Visualizes three layers of healthcare facilities with custom icons and detailed popups showcasing facility-specific properties.
  • Student and Airport Data: Integrates student and airport GeoJSON layers with stylized boundaries and popup details for detailed exploration.
  • Air Quality Index (AQI) Layer: - Fetches real-time AQI data for districts using the OpenWeatherMap API. - Displays pollutant concentrations (e.g., PM2.5, PM10) and AQI levels with intuitive markers.
  • Interactive Popups and Tooltips: - Clickable features provide in-depth information about districts, healthcare facilities and AQI levels.
  • Customizable Layers: - A layer control panel enables users to toggle between base maps and various overlays for tailored visualization.

This application is particularly suited for geospatial analysis, healthcare planning, environmental studies and decision-making processes requiring dynamic, layered map interactions.
